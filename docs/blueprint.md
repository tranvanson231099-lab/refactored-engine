# **App Name**: VietFlex Input

## Core Features:

- Vietnamese IME Core Logic: Implement essential Vietnamese input methods (Telex, VNI) to accurately convert Roman characters to Vietnamese with proper diacritics.
- Input Method Selector: Provide a simple interface within the extension popup to switch between Telex and VNI input methods.
- Extension Activation Toggle: Allow users to easily enable or disable the Vietnamese input functionality across webpages via the extension popup.
- Persistent Settings Management: Save and recall the user's preferred input method and activation status to maintain continuity across browser sessions.
- Smart Typing Suggestions: An AI-powered tool that provides intelligent word completions or suggests tone mark corrections based on linguistic context and common Vietnamese typing errors.

## Style Guidelines:

- Primary color: A vibrant, clear blue (#2E7AC2) to symbolize efficiency and precision.
- Background color: A very light, subtle blue-grey (#EBF1F6) for a clean, non-distracting canvas.
- Accent color: A lively turquoise (#33CCCC) to highlight interactive elements and call-to-actions, ensuring good contrast.
- Headline and body font: 'Inter' (sans-serif) for its modern, neutral, and highly readable design, suitable for a functional utility.
- Utilize minimalist and intuitive icons for toggles and settings, aligning with a clean and efficient user experience.
- Design a compact and responsive layout for the extension popup and any accompanying options pages, prioritizing ease of access and usability on Chrome OS Flex.
- Incorporate subtle animations for state changes (e.g., enabling/disabling the IME) to provide clear visual feedback without being intrusive.
