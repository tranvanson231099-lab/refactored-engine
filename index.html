CHƯA XONG ĐÂU BẠN ƠI! 🚀

Bạn đang nhìn thấy trang này vì bạn đã chọn thư mục gốc vietflex.

HÃY CHỌN THƯ MỤC "out" ĐỂ SỬA LỖI CSP
1
Gõ lệnh npm run build vào Terminal.
2
Vào chrome://extensions, xóa bản cũ.
3
Nhấn Tải tiện ích đã giải nén, tìm thư mục vietflex rồi chọn thư mục con tên là out.

(Thư mục out là thư mục chứa sản phẩm cuối cùng đã sạch lỗi)
