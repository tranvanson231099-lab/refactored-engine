BẠN ĐÃ CHỌN SAI THƯ MỤC!

Lỗi "CSP" hoặc "ERR_FILE_NOT_FOUND" xảy ra do bạn chọn thư mục gốc vietflex.

1. Chạy lệnh: npm run build

2. Vào Chrome Extensions, nhấn Tải tiện ích đã giải nén.

3. Tìm đến thư mục vietflex, sau đó CHỌN THƯ MỤC CON:
/vietflex/out

(Thư mục 'out' chứa mã nguồn sạch lỗi CSP)
